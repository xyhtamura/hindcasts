liner notes — un-live

Every realtime effect is causal: its output at time t can depend only on frames at or before t. A live feedback smear can therefore only trail behind motion, and seeking has to wipe its buffer — the state can't be rebuilt from the middle. Prolepsis drops causality on purpose. It reads the clip end-to-end, then runs the same field with the time-arrow free: smears that bloom ahead of an event, symmetric zero-phase trails, and a scrubber where every frame is already correct — so seeking is instant and acausal instead of a reset. The live framesmear isn't deleted; it's kept as the degraded, causal variant of the clean offline thing.

anticipation
The field run backward in time. Each transient gets a precursor swarm ramping up to it — pre-echo as deliberate texture. The video cousin of pre-verb.
symmetric
Forward + backward passes combined: smear blooms into and out of each event. Bidirectional, zero-phase. The balance slider weights precursor against wake.
transient-aware
A pre-analysis pass scores how big each upcoming change is; precursor density scales with what's coming. Multi-pass measurement no live system can do.
even exposure
Global statistics: luminance normalized against the whole clip's mean rather than a running estimate. Brightness held steady across the piece.
memory decay
Reverb time. How long the feedback tail persists — now in either temporal direction — before dissolving.
new frame opacity
How aggressively each fresh frame inscribes over the buffer. Lower = longer smear.
flow X / Y
Drift direction of the feedback buffer per frame. In anticipation/symmetric the backward pass mirrors it, so precursors converge into the event.
feedback zoom / rotation
Room size and spin. Above 1 the buffer expands; rotation accumulates spiral vortex trails.
oscillation
Tape wow. Flow direction breathes sinusoidally — unstable, warping accumulation.
strobe
Gated reverb. Decay pulses between near-frozen and fast-fade, stamping discrete echoes.
hue drift / chroma split
Shimmer and chromatic aberration, baked per frame so playback stays a pure blit.
edge / light / softening
The inscription mask: which parts of each frame get written hardest into the buffer, and how much the buffer diffuses between writes.
